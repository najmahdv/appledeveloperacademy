Project 01 · 2026 · Self-Initiated Innovation

🌊 SafeTide: Offline Empathic AI for Maritime Safety

Role: Solo Creator (Research, UX Design, AI Engineering)
Python Flask LLaMA 3 Ollama Docker Redis Figma
🔗 View on GitHub
🔍 Why I Built This

Maritime workers in remote coastal zones face extreme safety risks and mental health challenges, yet they lack internet access to seek help. Existing solutions assume connectivity, leaving the most vulnerable behind.

🛠️ How It Works

I engineered an offline AI chatbot that runs entirely on local hardware using Ollama (LLaMA 3). The conversation flows are designed to be highly empathetic, ensuring workers feel safe and heard when reporting incidents anonymously.

🎯 What I Delivered (My Contribution)

As the sole developer and designer, I handled the entire product lifecycle:
• Conducted user research based on maritime government data & published reports.
• Designed the complete UI/UX & anonymous reporting flow in Figma with user personas & empathy maps.
• Developed the backend architecture using Python + Flask + Redis.
• Containerized the system with Docker for seamless offline installation.
• Advanced to the national finals of the Youth ESG Maritime Innovation Challenge 2026 — organized by Kemlu RI × Sampoerna University × Blue Ocean Strategy Fellowship™.

Project 02 · 2025 · Academic Final Project (Tugas Besar)

📊 SDA Bakarun: Bakery Data Management System

Role: Solo Developer (Architecture, C++ Logic, Python Automation)
C++ Borland C++ Builder 6 Linked Lists Data Structures Python (Automation)
🔍 Why I Built This

Local bakeries struggle with complex stock management and revenue recapitulation. I needed to build a highly efficient, lightweight system from scratch to truly understand data optimization at its core.

🛠️ How It Works

Built purely in C++ and Borland Builder 6, the system uses raw singly linked lists for dynamic memory, implementing custom bubble sort and search algorithms for real-time transaction management.

🎯 What I Delivered (My Contribution)

I architected and coded the entire full-stack application independently:
• Designed a scalable C++ linked-list data structure for inventory & transactions.
• Built a fully functional GUI using Borland C++ Builder 6.
• Going Beyond: Wrote custom Python automation scripts to auto-generate the 100+ page project appendix AND PowerPoint presentation directly from the C++ source code — demonstrating a hybrid cross-technology mindset.

💻 Application Screenshots (Scroll →)

"This project deepened my appreciation for clean architecture and algorithmic thinking. Going beyond the assignment by building Python automation tools showed me how different technologies can complement each other. A C++ project became more powerful with Python scripting on top. This cross-technology mindset is exactly what I want to develop further at the Apple Developer Academy."

🔗 Connection to Apple Academy

The Apple Developer Academy emphasizes Challenge-Based Learning (CBL). This project was my first experience with that approach: I identified a real challenge, investigated solutions through data structure theory, and acted by building a complete working system. The CBL framework now feels natural to me because I've already lived it.
